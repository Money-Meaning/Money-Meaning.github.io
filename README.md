# Money Talks: An Economics Blog (Jekyll scaffold)

This repo contains a starter Jekyll configuration for a finance/economics blog.

Quick start:

1. Install Ruby + Bundler (if not already installed)
   - ruby --version
   - gem install bundler

2. Install dependencies
   - bundle install

3. Serve locally
   - bundle exec jekyll serve
   - Open http://127.0.0.1:4000 (or http://localhost:4000)

Notes:
- Replace "Your Name" in _config.yml with your display name and update social handles.
- If deploying to GitHub Pages under a repository (e.g. Money-Meaning/economic-blog), set `url:` and `baseurl:` appropriately.
- The config enables jekyll-feed and jekyll-seo-tag; add `{{ site.seo }}` to your layouts/head and link to `/feed.xml` if needed.
